A Smart Queue Management System built using HTML, CSS, JavaScript with queue logic inspired from C programming. The system simulates customer queues, waiting time tracking, and animated GUI visualization.


# Smart Queue Management System

This project is a Queue Management Simulator that visualizes how a queue works in real time.

The system allows users to add and remove customers while automatically tracking waiting time.

## Features

- Add Customer to Queue
- Remove Customer from Queue
- Automatic Waiting Time Counter
- Queue Visualization
- Statistics Panel
- Animated Background Objects
- Modern Glass UI

## Technologies Used

Frontend
- HTML
- CSS
- JavaScript

Backend Logic
- C Programming (Queue Data Structure Concept)
